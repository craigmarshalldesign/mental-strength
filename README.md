# Craig Marshall Design - Mental Strength Guides

This repository contains guides for mental strength and personal growth by Craig Marshall Design.

## Website Structure

- `index.html` - Main landing page with links to guides
- `guides/` - Directory containing all guides
  - `mastering-indifference.html` - Guide on mastering emotional indifference

## Setting Up GitHub Pages

To deploy this site using GitHub Pages:

1. Push this repository to GitHub
2. Go to the repository on GitHub
3. Click on "Settings"
4. Scroll down to the "GitHub Pages" section
5. Under "Source", select "main" branch
6. Click "Save"
7. Your site will be published at `https://[your-username].github.io/[repository-name]/`

## Adding New Guides

To add a new guide:

1. Create a new HTML file in the `guides/` directory
2. Follow the same styling and structure as existing guides
3. Add a card for the new guide in `index.html` by duplicating the existing guide card and updating content

## Local Development

To preview the site locally, you can use any local server. For example:

- Using Python 3: `python -m http.server`
- Using Python 2: `python -m SimpleHTTPServer`
- Using Node.js: Install `http-server` via npm and run `http-server`
